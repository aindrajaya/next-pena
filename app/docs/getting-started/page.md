## Installation
The first step to getting started with create-pena-app is to install the CLI tool on your system. Follow these instructions:

  ### Step 1: Installing dependencies

  Install Node.js and npm: Make sure you have Node.js and npm (Node Package Manager) installed on your system. You can download and install Node.js from nodejs.org.

  ```shell
  npm -v
  npm install -g create-pena-app-ext
  ```

  ### Step 2: Create Pena app Scaffolding

  Create a new project using create-pena-app:
  ```shell
  create-pena-app my-docs-platform
  cd my-docs-platform
  ```
